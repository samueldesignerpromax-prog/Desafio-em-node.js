# Gerador de QR Code para E-commerce

## Descrição

Este projeto consiste em um gerador de QR Codes executado diretamente no terminal utilizando Node.js.

A ferramenta permite gerar QR Codes rapidamente para links de produtos de e-commerce, facilitando o compartilhamento e acesso direto às páginas de vendas.

---

## Tecnologias utilizadas

- Node.js
- Biblioteca qrcode-terminal

---

## Instalação

Clone o repositório:

```
git clone https://github.com/seuusuario/node-qrcode-generator
```

Entre na pasta:

```
cd node-qrcode-generator
```

Instale as dependências:

```
npm install
```

---

## Execução

Execute o programa:

```
node index.js
```

Digite o link do produto quando solicitado.

O QR Code será exibido diretamente no terminal.

---

## Exemplo de uso

Entrada:

```
https://loja.com/produto123
```

Saída:

QR Code gerado no terminal.
